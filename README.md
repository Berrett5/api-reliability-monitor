# API Reliability Monitor

API Reliability Monitor is a Java-based monitoring tool that tracks the health and performance of API endpoints.  
The system periodically checks configured APIs, records response times and status codes, and stores monitoring results in a MySQL database.

This project demonstrates backend development, database design, and system monitoring concepts.

---

## Features

- Monitor API endpoints
- Measure response times
- Detect failures
- Record HTTP status codes
- Store monitoring history in MySQL
- Calculate uptime percentages
- Display monitoring data in a dashboard (planned)
